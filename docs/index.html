Introduction

irispreppy is built to perform radiometric calibration and point spread function deconvolution of data from the spectrograph onboard the Interface Region Imaging Spectrograph (IRIS).

The radiometric calibration follows the procedure laid out in section 5.2 of IRIS Technical Notes 26 (ITN 26; html; pdf), and more generally IRIS Technical Notes 24 (ITN 24; pdf).

The point spread function deconvolution is done using either the Richardson-Lucy deconvolution method, or through division in Fourier space. The number of default Richardson-Lucy iterations used is discussed in Courrier et al. (2018).
